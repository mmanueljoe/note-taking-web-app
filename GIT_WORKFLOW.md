## Git Commands Used

- `git status` – Used to check current changes and branch status
- `git branch` – Used to view existing branches
- `git branch <name>` – Used to create new branches
- `git checkout <branch>` – Used to switch between branches
- `git checkout -b <branch>` – Used to create and switch to new feature branches
- `git add <file>` / `git add .` – Used to stage changes before committing
- `git commit -m "type: message"` – Used to record changes in history
- `git log --oneline --graph --all` – Used to visualize commit and branch history
- `git merge <branch>` – Used to merge a branch into the current one
- `git pull` / `git pull --rebase origin <branch>` – Used to sync with remote
- `git push origin <branch>` – Used to publish local commits to GitHub
